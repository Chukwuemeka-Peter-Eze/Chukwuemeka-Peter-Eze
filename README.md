## About My Tech Stack
I am a Cloud and DevOps Engineer, building a strong foundation across the essential tools and technologies that power modern infrastructure. My experience spans cloud platforms (AWS, Azure, GCP), infrastructure as code (Terraform, CloudFormation), containerization (Docker, Kubernetes, Helm), automation (Ansible, Bash, Python), and CI/CD pipelines (Jenkins, GitHub Actions, GitLab, CircleCI).

I actively design and implement hands-on projects that go beyond theory, focusing on real-world, enterprise-grade use cases—from scalable deployments to monitoring and observability with Prometheus, Grafana, ELK Stack, and Datadog.

I am passionate about building resilient, automated systems and continuously expanding my expertise to meet the evolving challenges of modern DevOps.
